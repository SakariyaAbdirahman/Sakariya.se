Overview

A multiplayer TicTacToe game developed inside Android studio using Kotlin, Jetpack Compose and Firebase as a Database.

Key Features

  • Player creation.
  • Overview of online players you can challange.
  • Accept/Decline challanges.
  • Play a game of TicTacToe using 3x3 grid.
  • Winner screen with higlighted winnig move.
